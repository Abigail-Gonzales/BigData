# Steam Nexus: Recomendador Evolutivo de Videojuegos

Este proyecto utiliza el ecosistema de Steam para explorar la descubrimiento de dominios, sistemas de recomendación y analítica de grafos.

## Estructura del Proyecto

- `data/`: Almacena los conjuntos de datos en sus diferentes etapas (raw, interim, processed).
- `notebooks/`: Cuadernos de Jupyter para exploración y análisis rápido.
- `src/`: Scripts de Python para pipelines de datos y modelado.
- `reports/`: Documentación técnica e informes de hitos.
- `artifacts/`: Modelos guardados, visualizaciones y otros archivos generados.

## Requisitos

Para instalar las dependencias necesarias, ejecuta:

```bash
pip install -r requirements.txt
```

## Reproducibilidad

Consulta el `runbook` (próximamente) en la carpeta `reports/` para instrucciones detalladas sobre cómo ejecutar el pipeline.
